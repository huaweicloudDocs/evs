# 查询云硬盘过户记录列表详情<a name="evs_04_3074"></a>

## 功能介绍<a name="zh-cn_topic_0092902037_section44805042171914"></a>

查询当前租户下所有云硬盘的过户记录列表的详细信息，比如过户记录创建时间、ID以及名称等信息。
