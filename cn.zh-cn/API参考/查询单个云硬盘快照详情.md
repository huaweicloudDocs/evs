# 查询单个云硬盘快照详情<a name="evs_04_2019"></a>

## 功能介绍

查询单个云硬盘快照信息。支持企业项目授权功能。

## 调试<a name="atuogenerate_1"></a>

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=EVS&api=ShowSnapshot)中调试该接口，支持自动认证鉴权。API Explorer可以自动生成SDK代码示例，并提供SDK代码示例调试功能。

## URI

GET /v2/\{project\_id\}/cloudsnapshots/\{snapshot\_id\}
