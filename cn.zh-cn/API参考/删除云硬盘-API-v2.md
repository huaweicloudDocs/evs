# 删除云硬盘<a name="evs_04_2008"></a>

## 功能介绍<a name="section10932885"></a>

删除一个云硬盘。支持企业项目授权功能。

## URI<a name="section31287108"></a>

-   URI格式

    DELETE /v2/\{project\_id\}/cloudvolumes/\{volume\_id\}
