# 查询云硬盘列表（废弃）<a name="evs_04_2012"></a>

## 功能介绍<a name="section39223476"></a>

>![](public_sys-resources/icon-notice.gif) **须知：**   
>该接口已废弃，请使用性能更佳的接口，具体请参见[查询云硬盘列表](查询云硬盘列表-Cinder-v2.md)。
